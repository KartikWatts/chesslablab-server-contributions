# /restart

## Description

Restarts an existing game.

| Name | Description | Required |
| ---- | ----------- | -------- |
| `hash` | The unique hash of the game. | Yes |

## Usage

### Example

```js
ws.send('/restart ffc536a8f44fc21b4d254e4fb85d7e33');
```
